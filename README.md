# Air Quality Monitoring System

## Overview
This project is an Arduino-based air quality monitoring and alert system using the MQ135 gas sensor. The system continuously monitors indoor air quality and displays real-time values on an LCD display.

## Features
- Real-time air quality monitoring
- MQ135 gas sensor integration
- LCD display output
- LED indication for air quality levels
- Buzzer alert for poor air quality

## Components Used
- Arduino UNO
- MQ135 Sensor
- LCD Display
- LEDs
- Buzzer

- Hospital room monitoring
- Indoor pollution detection
